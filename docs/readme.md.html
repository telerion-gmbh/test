Introduction

The telerion SDK provides fast and easy way for iOS application to enable telerion services which include messaging, voice and video, click-to talk, ICC, AI bots, etc.

The SDK entry point is the class [TSDKCore] which includes the following functions:

  • Login to telerion server (or make authentication)
  • Logout
  • Make outgoing call
  • Make outgoing chat
  • Fire login/logout events
    Coming soon:
  • Register for push notifications to receive notification on incoming call or message
  • Handle push notifications
  • Process incoming calls
  • Process incoming chats

Example how to make an outgoing call:

var sdk: TSDKCore?
var session : TSDKSession?

sdk = TSDKCore("https:/fuse.telerion.com/")
sdk.login("1234-5678-9012-3456")
session = sdk.createSession(.call)
session!.connect("+18449255329")
...

The class [TSDKSession] represents an outgoing or incoming call or chat session. Possible session types are in the enum [TSDKSessionType].

The class [TSDKSession] includes various methods and events to manage active call or chat: mute, send DTMF, etc. See more details in our [Classes] description.

Introduction (Android)

The telerion SDK provides fast and easy way for Android application to enable telerion services which include messaging, voice and video, click-to talk, ICC, AI bots, etc.

The SDK entry point is the class [Core] which includes the following functions:

  • Login to telerion server (or make authentication)
  • Logout
  • Make outgoing call
  • Make outgoing chat
  • Fire login/logout events
    Coming soon:
  • Register for push notifications to receive notification on incoming call or message
  • Handle push notifications
  • Process incoming calls
  • Process incoming chats

Example how to make an outgoing call:

import com.telerion.androidsdk.Core;  
import com.telerion.androidsdk.CoreListener;  
import com.telerion.androidsdk.ChatMessage;  
import com.telerion.androidsdk.CompletionHandler;  
import com.telerion.androidsdk.Session;  
import com.telerion.androidsdk.SessionListener;  
import com.telerion.androidsdk.SessionType;  
import com.telerion.androidsdk.TelerionError;

Core mCore;
Session session;

mCore = Core.create(getApplicationContext(), "https:/fuse.telerion.com/");
mCore.setListener(this);
mCore.login("1234-5678-9012-3456");
mCore.createSession(type, new CompletionHandler<Session>() {  
    @Override  
	  public void onSuccess(Session session) {  
        if (null != ownNumber)  
           session.setOwnNumber(ownNumber);  
        session.setListener(MainActivity.this);  
        session.connect("+18449255329");  
      }  

	@Override  
	  public void onError(TelerionError e) {  
        Log.e("MainActivity", "Unable to create session: " + e);  
      }
}

The class [Session] represents an outgoing or incoming call or chat session. Possible session types are in the enum [SessionType].

The class [Session] includes various methods and events to manage active call or chat: mute, send DTMF, etc. See more details in our Classes description.
